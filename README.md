# Azimuth Predictions


## About
This repository includes a collection of scripts for annotating single cell RNA-seq query datasets from different data portals. Currently, the pipeline has been/is being set up for the following collections:

- [HuBMAP Kidney](hubmap-kidney)
- [NeMO Archive](nemo-archive)
- [UCSC Cell Browser Data](ucsc)
